How to use the Rename Dialog

1) Select all cards you want to rename by clicking the card names (multiple selection works with shift/ctrl) or 'select all'

2a) Select a pattern from the already defined pattern list (or modify/delete/add new patterns)

or

2b) Enter your own regexp / replace combination. The regexp input defines which parts of the card names should be matched. To use specific parts wrap them into matching groups (e.g (.*) matches the whole name). You can now use the replace input to define how the new name should look like: $1-9 is the matched group from the regex; $i a running index from top to down.

3) Apply changes to all selected cards by pressing the rename button
